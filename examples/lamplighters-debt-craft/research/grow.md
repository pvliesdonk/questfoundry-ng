### audience-and-access/localization_considerations.md#Common Mistakes

### Hardcoded Text Text embedded in code rather than externalized. **Fix:** All player-facing text in resource files. ### Concatenation Building sentences from fragments. **Fix:** Complete sentences with placeholders. ### Assuming English Rules Plural, gender, word order assumptions. **Fix:** Use localization-aware systems. ### No Context Translators working blind. **Fix:** Provide context for every string. ### Machine Translation Only Google Translate as final product. **Fix:** Professional translation with proper review. ### Ignoring Cultural Issues Direct translation without cultural consideration. **Fix:** Cultural adaptation alongside translation. ### No Budget/Time for Localization Treating localization as free afterthought. **Fix:** Budget time and money from project start. ---
